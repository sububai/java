3、绝对不要清除对象
  3.1、作用域（scope）由花括号的位置决定
    c/c++能将一个变量“隐藏”在一个更大的作用域，但是Java不可以
    {
      int x = 12;
      {int x = 96;//在Java中不合法}
    }
  3.2、对象的作用域
    Java对象不具备和主类型一样的存在时间，它会超出作用域的范围
    句柄和主类型一样，因为都存储在堆栈中
